====================================================

1. User the `pathology_lab.sql` to setup the db

Once done you'll have 2 operators

	Operators
	1. egg@wall.com dragons4eva
	2. azorahai@ptwp.com rpluslequalsj

We'll use them later.

2. Edit the `application/config/config.php` to set `base_url` on line 26

3. Edit the `application/config/database.php` to set up database details

4. Now go to http://<domain>/public e.g. mine was at http://localhost/pathology_lab/public/

That should get you up and running
